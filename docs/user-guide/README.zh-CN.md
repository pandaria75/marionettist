## 当前命名说明

当前面向用户的表面名称已经统一使用 **Marionettist**：

- CLI 命令，例如 `marionettist init`
- 配置文件，例如 `marionettist.config.yaml`
- 受管状态文件，例如 `.marionettist/manifest.json`

部分开发文档或迁移文档仍会提到历史上的 **harness** 命名，因为它们描述的是旧行为或迁移步骤。
